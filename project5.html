Wedding Banquet Table Planner

Skills: JavaScript | Bootstrap


Click here for the app!

I used JavaScript to create a function that takes the inputted amount of guests and tables in order to calculate the number of guests that will be evenly distributed to each table. I practiced using "if" statements and worked with several variables to print out the number of guests at each table. Through this project, I became more familiar with using the operator modulo (%) and using Bootstrap to create alerts.
